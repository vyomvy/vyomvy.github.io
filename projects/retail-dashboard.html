Pipeline 03 / Retail

Retail sales analysis dashboard

An interactive Power BI dashboard built to separate real profit engines from cities that just look busy.

Source code ↗
The problem

Retail sales data across 8 Indian cities needed to go beyond a static report — the business wanted to filter and explore live, not wait for a new export every time a question came up.

The approach
01
Custom DAX measures

Built measures for Average Order Value, profit margin, and category-wise performance.

02
City & category slicers

Added live filters so anyone can explore the data themselves instead of requesting a new cut.

03
Interactive dashboard

Assembled everything into a single Power BI view built for exploration, not just reporting.

Key findings
8

Cities compared side by side on AOV, margin, and category performance in one view.

DAX

Custom measures expose profit-margin gaps that raw revenue numbers alone hide.
